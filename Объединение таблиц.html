merge позволяет соединять таблицы не только по индексам, но и по полям

In [19]:
left_data = pd.DataFrame({"key1": ["a", "b", "c"], "key2": ["x", "y", "z"], "lval1": [0, 1, 2]})
left_data
Out[19]:
key1 key2 lval1
0 a x 0
1 b y 1
2 c z 2
In [20]:
right_data = pd.DataFrame({"key1": ["a", "b", "c"], "key2": ["x", "a", "z"], "lval1": [6, 7, 8]})
right_data
Out[20]:
key1 key2 lval1
0 a x 6
1 b a 7
2 c z 8

По умолчанию merge соединяет таблицы через inner
